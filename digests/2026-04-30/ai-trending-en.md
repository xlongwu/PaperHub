**Terminal-as-Agent-Platform** is the most surprising new direction. [Warp](https://github.com/warpdotdev/warp)'s 12,822-star day demonstrates that developers want their *existing environment* augmented, not replaced. This contrasts with IDE-centric tools like Cursor and suggests a bifurcation: terminal-native for power users, IDE-integrated for mainstream. The "agent harness" category—[jcode](https://github.com/1jehuang/jcode), [everything-claude-code](https://github.com/affaan-m/everything-claude-code)—represents emergent infrastructure for standardizing agent execution environments.

**Voice AI's Open-Source Moment** arrives with [microsoft/VibeVoice](https://github.com/microsoft/VibeVoice), timed as frontier voice models (likely GPT-5 voice, Gemini Live expansion) reach maturity. Microsoft's open-source strategy here mirrors their Phi/Mistral approach: commoditize the modality, monetize the platform.

**China Model Ecosystem Maturation** is evident in [CJackHwang/ds2api](https://github.com/CJackHwang/ds2api)'s API standardization layer for DeepSeek, and Ollama's inclusion of Kimi-K2.5, GLM-5, MiniMax. The technical barrier is shifting from model access to *interoperability*.

**MCP (Model Context Protocol) is becoming plumbing**, not headline feature—[activepieces/activepieces](https://github.com/activepieces/activepieces)'s 400+ MCP servers, [zilliztech/claude-context](https://github.com/zilliztech/claude-context)'s code search MCP.
