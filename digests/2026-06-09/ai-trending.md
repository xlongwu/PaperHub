## 三、趋势信号分析

**Agent 技能化（Agent Skillification）** 正成为 2026 年 Q2 最强劲的爆发方向。今日 Trending 前三席中，[last30days-skill](https://github.com/mvanhorn/last30days-skill)、[google/skills](https://github.com/google/skills)、[pm-skills](https://github.com/phuryn/pm-skills) 均围绕"Skill"概念构建，标志着 Agent 生态从"造框架"进入"攒能力"的新阶段——开发者不再重复造轮子，而是封装可复用的垂直能力模块。Google 与 OpenAI 同日开源 Skills/Plugins 体系，巨头生态标准之争浮出水面。

**零成本/本地优先** 成为开发者核心诉求。[Agent-Reach](https://github.com/Panniantong/Agent-Reach) 的"zero API fees"定位、[whichllm](https://github.com/Andyyyy64/whichllm) 的硬件适配基准、[goose](https://github.com/aaif-goose/goose) 的 Rust 本地执行，共同指向一个趋势：在大模型 API 成本波动与数据隐私焦虑下，社区正在构建"去中心化"的 Agent 运行环境。Rust 技术栈凭借性能与安全性，在向量索引（turbovec）、Agent 运行时（goose）、向量数据库（Qdrant）等底层设施中加速渗透。

与近期行业动态的关联：Claude Code 的持续热度催生了 [learn-claude-code](https://github.com/shareAI-lab/learn-claude-code)、[claude-howto](https://github.com/luongnv89/claude-howto)、[claude-mem](https://github.com/thedotmack/claude-mem) 等衍生生态；MCP（Model Context Protocol）协议在 [activepieces](https://github.com/activepieces/activepieces) 等项目中实现规模化集成，正从概念走向生产。
